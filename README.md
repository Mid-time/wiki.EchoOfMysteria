## 关于本维基

**欢迎，探寻回响之人。**  
  
这是一部由探索者、学者与冒险者共同编纂的活态知识库，献给《神秘回响》——那个建立在旧日伤痕之上、由齿轮、信仰、鲜血与秘密编织而成的世界。  
  
在这里，蒸汽机的轰鸣与古代魔法的低语在同一座城市的血脉中流淌；学者的理性思索与星空深处不可名状的凝视共享同一片夜空。这里是新伦敦，是索多玛，是利米亚，是沉入深海的旧伦敦，是悬浮于天际的连云港——是所有已知与未知交汇的边界。  
  
本维基旨在为每一位踏入这片土地的旅人提供指引：从角色的诞生到飞升，从街巷的传说到深渊的秘密，从刀剑的碰撞到星空的低语。所有知识，无论危险与否，皆在此记录——因为我们相信，铭记是抵抗遗忘的唯一方式。  
  
## 世界核心概念

· 大沉降：约三百年前，整个旧伦敦沉入深海。这场灾难塑造了当今世界的格局，其涟漪至今仍在政治、科技与信仰的脉动中回荡。  
· 罪孽与飞升：八种原罪编织着凡人的灵魂。直面它们，或在磨炼中超越，或在沉沦中异变——飞升之路诱人而危险。  
· 二十四主教会：每一时辰皆由一位司辰守护。教会不仅是信仰的殿堂，更是影响世界进程的庞大势力。  
· 金枝：一个“不应存在”的概念。接触它，意味着触碰世界的本质，也意味着承受不可知的代价。
